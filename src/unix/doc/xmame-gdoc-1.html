1. Einleitung und Links

1.1 Alles immer up-to-date halten !

Die aktuellste Version dieser Anleitung, die Sourcen von XMame und vieles mehr kann auf der Homepage des XMame-Projektes gefunden werden unter http://x.mame.net/

1.2 Einige Links

  • Die offizielle Hompage von M.A.M.E. ist http://www.mame.net
  • Grustibus, ein sehr gutes Frontend (wenn Du Gnome kennst) http://grustibus.sourceforge.net/
  • Gnomame, ein anderes grossartiges Gnome-basiertes Frontend http://gnomame.sourceforge.net/
  • Tkmame, das Tcl/TK Frontend dass unter Linux/Unix und MS Windows funktioniert http://tkmame.retrogames.com/
  • MAME Action Replay Page für die Leute, die einen Blick auf andere Monitore werfen wollen http://marp.retrogames.com/
  • und natürlich nicht unsere Projekt Homepage vergessen http://x.mame.net/
